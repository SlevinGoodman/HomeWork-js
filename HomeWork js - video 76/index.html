Способ создания переменных через let позволяет позже изменить значение переменной. Применяется только в случае обоснованной необходимости.

Способ создания переменных через const не позволяет позже изменить значение переменной. Такой способ является более приоритетным, чем let, во избежание возникновения ошибок.
